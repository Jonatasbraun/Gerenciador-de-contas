### Central de Contas Trans Helo - Empacotamento ###

1. Execute o comando abaixo para gerar o executável:
pyinstaller --noconfirm --onefile --windowed --icon=icone.ico --name="CentralTransHelo" saldo_contas_gui.py

2. Após gerar o .exe, abra o instalador_transhelo.iss no Inno Setup
3. Compile para gerar o instalador final.
